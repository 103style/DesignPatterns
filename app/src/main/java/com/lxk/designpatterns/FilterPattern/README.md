# Filter Pattern 过滤器模式（结构型模式）
过滤器模式 或 标准模式（Criteria Pattern）是一种设计模式，
这种模式允许开发人员使用不同的标准来过滤一组对象，通过逻辑运算以解耦的方式把它们连接起来。
这种类型的设计模式属于 结构型模式，它结合多个标准来获得单一标准。
